Web-based Interactive River Model (WIRM)

A simple water quality model for understanding the effects of pollution on dissolved oxygen levels in rivers and streams. This site was created as a demonstration of modern web application development, and how new technologies can be used to create interactive, easy-to-use environmental simulation models. It is intended primarily for educational purposes, but can be used for the analysis of real systems.
