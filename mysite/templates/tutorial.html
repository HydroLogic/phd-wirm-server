Introduction

Before using WIRM, this tutorial will walk you through the steps for properly using the model to better understand how best to use it.

Setting Up a New Project

The first step is to set up a new project. A project is a defined as an application of the model to a specific river or stream under specific conditoins. For example, one project may represent the Mississippi River under low flow conditions when a wastewater treatment plant is discharging a typical load. Another project could represent the same river under different conditions such as high flow and reduced wastewater discharge.
